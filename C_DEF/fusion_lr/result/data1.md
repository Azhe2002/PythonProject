# data1 逻辑回归融合权重 — 训练结果

## 数据集

- **来源**: Kaggle Real and Fake Face Detection
- **训练集**: 2041 张 (real=1081, fake=960)
- **无验证/测试拆分**，采用 5 折交叉验证

## 四分支分数分布

| 分支 | real 均值 | fake 均值 | 差异 |
|---|---|---|---|
| 物理噪声 | 0.4628 | 0.4553 | +0.0075 |
| 频域残差 | 0.8270 | 0.8290 | -0.0020 |
| RGB 语义 | 0.9524 | 0.9499 | +0.0025 |
| 人脸光滑度 | 0.6897 | 0.7051 | -0.0154 |

real/fake 差异均在 0.02 以内，两类在四维特征空间中高度重叠。

## 交叉验证

| C | mean AUC |
|---|---|
| 0.001 | 0.5270 |
| 0.01 | 0.5290 |
| 0.1 | 0.5307 |
| 1.0 | 0.5309 |

最优 C = 1.0

## 性能对比

| 指标 | 手工基线 | 逻辑回归融合 |
|---|---|---|
| AUC | 0.5036 | 0.5409 |
| F1 | 0.0062 | 0.3087 |
| Accuracy | 0.5311 | 0.5282 |
| ECE | 0.1256 | 0.0085 |

## 权重对比

| 分支 | 手工权重 (和=1) | LR 原始系数 | LR 相对重要性 (和=1) |
|---|---|---|---|
| 物理噪声 P | 0.460 | -1.371 | 0.346 |
| 频域残差 F | 0.320 | -1.389 | 0.351 |
| RGB 语义 R | 0.100 | -0.293 | 0.074 |
| 人脸光滑度 S | 0.120 | 0.905 | 0.229 |
| 截距 | — | -0.120 | — |

**相对重要性** = \|w_i\| / sum(\|w\|)，始终非负且和为 1。原始系数的符号表示方向：正值→推高真实概率，负值→推高 AI 概率。

## 结论

**AUC≈0.5，四分支特征在 data1 上无判别力。** 该数据集不适合用当前手工特征管道进行检测。data2（FFHQ+StyleGAN2，高分辨率原始质量）是更合适的验证目标。
