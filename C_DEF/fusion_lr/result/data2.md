# data2 逻辑回归融合权重 — 训练结果

## 数据集

- **来源**: Kaggle 140k Real and Fake Faces (FFHQ + StyleGAN2)
- **训练集**: 100,000 张 (real=50,000, fake=50,000)
- **验证集**: 20,000 张 (real=10,000, fake=10,000)
- **测试集**: 20,000 张 (real=10,000, fake=10,000)

## 权重对比

| 分支 | 手工权重 (和=1) | LR 原始系数 | LR 相对重要性 (和=1) |
|---|---|---|---|
| 物理噪声 P | 0.460 | -1.970 | 0.241 |
| 频域残差 F | 0.320 | 0.146 | 0.018 |
| RGB 语义 R | 0.100 | 4.781 | 0.584 |
| 人脸光滑度 S | 0.120 | 1.291 | 0.158 |
| 截距 | — | -0.001 | — |

**相对重要性** = \|w_i\| / sum(\|w\|)，始终非负且和为 1，衡量每个分支对模型决策的贡献大小。原始系数的符号表示方向：正值→推高真实概率，负值→推高 AI 概率。物理噪声分支原始系数为负，与该分支的理论预期矛盾。

## 结论

**AUC≈0.57，四分支特征在 data2 上仅有微弱判别力。** 逻辑回归将 AUC 从 0.556 提升至 0.571（+0.015），但远未到实用水平。关键问题：

1. **分支分数 real/fake 差异过小**（均 < 0.015），特征空间两类高度重叠
2. **物理噪声分支权重反转**（学习为负值），与该分支的理论预期矛盾
3. **RGB 语义成为主导分支**（权重 0.584），但其区分力有限
4. **手工权重完全失效**（F1=0），说明 0.46/0.32/0.10/0.12 的经验估计与数据分布不符

## 根本原因分析

当前四分支评分管道是为真实相机拍摄 vs AI 生成设计的（依赖传感器噪声、CFA 周期等物理特征），但 data2 中的 StyleGAN2 图像经过后处理/压缩，物理噪声特征可能已被破坏。此外，FFHQ 真实图像本身也经过了预处理（对齐、裁剪），减弱了传感器噪声信号。

**方案一（逻辑回归融合）的上限受限于四分支特征的表达能力，需要在特征层面或模型层面升级（方案三 deep CNN）。**
